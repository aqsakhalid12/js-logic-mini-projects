# JS Logic Mini Projects

This repository contains **5 JavaScript mini projects** that demonstrate basic programming concepts such as loops, conditions, functions, and user input.  
These projects are designed for practice and educational purposes.

---

## 📌 Projects Included

1. **Grocery Bill Calculator** – Calculates total bill and applies discount if the total exceeds a certain amount.  
2. **Student Result System** – Computes total marks, average, and grade for a student.  
3. **ATM Withdrawal System** – Simulates an ATM with balance check and withdrawal attempts.  
4. **Restaurant Order System** – Calculates order total and applies discount for large bills.  
5. **Employee Salary Calculator** – Computes salary including overtime for multiple employees.

---

## 🚀 How to Run

1. Open `index.html` in any modern browser.  
2. Open the browser console (F12 → Console).  
3. Follow the prompts in each program to enter input.  
4. Results will appear in the console.


## 💡 Concepts Used

- Functions (`function`)  
- Loops (`for`)  
- Conditional statements (`if-else`)  
- Arithmetic operators (`+`, `-`, `*`, `/`)  
- User input (`prompt()`)
